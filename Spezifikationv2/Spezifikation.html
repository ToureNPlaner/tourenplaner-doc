Introduction

The protocol is using HTTP/S as Transport Layer. The encoding used is JSON and optionally SMILE (binary JSON) as used by the Jackson project. The server might sent answers in SMILE if the client adds application/x-jackson-smile to its HTTP Accept Header, plain JSON otherwise.

The server supports two modi: public and private. In public mode only the info and alg* request are supported. In private mode user authentication is used. The following errors can arise in private mode from every alg* request and private request: "EAUTH - 401 Unauthorized - Wrong username or password", "ENOTVERIFIED - 403 Forbidden - User account is not verified" and "EDATABASE - 500 Internal Server Error - The server cannot contact its database or a database error occurred". The following error can arise in public and private mode: "EUNKNOWNURL - 404 Not Found - An unknown URL was requested"

Data Formats

Coordinates

All coordinates are sent as integers using a fixed point presentation with a factor of 10e7. E.g. the latitude 50.1234 is encoded as 5012340000.

Date-Time-Strings

Date-Time-Strings are encoded after ISO 8061. E.g.: 2000-12-11T13:55:30.000+0000 or 2000-12-11T13:55:30Z.

/registeruser

Description

Used to register new useraccaounts. Needs e-mail, password, prename, surname and address (optional the admin flag). If you sent an user for authentification, it should be an admin else nothing will be done. If you only want to register you should not sent any authentification. This registered user needs then a registration and has the flag "need_verification" only.

Errors

Error ID Http Statuscode Message and possible Details

EAUTH

401 Unauthorized

Wrong username or password

EBADJSON

400 Bad Request

Could not parse JSON object or JSON object is missing
- Content is empty
- JSON user object was not correct (needs email, password, firstname, lastname, address)
- JSON user object was not correct ("admin" should be boolean)
- JSON user object was not correct ("status" was not a valid value)

EBUSY

503 Service Unavailable

This server is currently too busy to fulfill the request

ECOMPUTE

400 Bad Request

The computation could not be completed

EDATABASE

500 Internal Server Error

The server cannot contact its database or a database error occurred

EINTERNAL

500 Internal Server Error

An internal server error occurred
- Unspecified server exception

ELIMIT

400 Bad Request

The parameter limit is missing or invalid
- The parameter limit is missing
- The given parameter limit is not a natural number (positive or zero)

ENOREQUESTID

404 Not Found

The parameter request id is missing, invalid or unknown to this server
- The given parameter id is not in the database
- The parameter id is missing
- The given parameter id is not a natural number (positive or zero)

ENOTADMIN

403 Forbidden

You are not an admin
- You cannot view requests or responses of other users
- You must be admin to list users
- You must first logout yourself to register users if you are not an admin.
- You must be admin if you want to use the user id parameter

ENOTVERIFIED

403 Forbidden

User account is not verified

ENOUSERID

404 Not Found

The parameter user id is missing, invalid or unknown to this server
- The given parameter id is not in the database
- The parameter id is missing
- The given parameter id is not a natural number (positive or zero)
- The given parameter id is not all or a natural number (positive or zero)

EOFFSET

400 Bad Request

The parameter offset is missing or invalid
- The parameter offset is missing
- The given parameter offset is not a natural number (positive or zero)

EREGISTERED

403 Forbidden

This email is already registered

EUNKNOWNALG
